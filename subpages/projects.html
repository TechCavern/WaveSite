WaveTact - Linux IRC Bot - MIT License

WaveTact is an IRC bot built upon the PircBotX framework. This bot is intended as an all-in-one utility bot, but NOT a replacement for IRC services. Rather it is intended to compliment IRC services. NickServ authentication is recommended, but the ability to auth by the bots own authentication system as well as by nick only is available as well. This bot has built-in support for inspircd, unrealircd and charybdis as well as anope and atheme services. Forks of any of these should work but is not guaranteed.
